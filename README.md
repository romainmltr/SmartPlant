# SmartPlant

## :dart: About ##

This project is a smart plant who can give you data for your plant 


## :sparkles: Features ##

:heavy_check_mark: Alert When your plant need water\
:heavy_check_mark: Give water to your plant ;\
:heavy_check_mark: Info of humidity ;\

## :rocket: Technologies ##

 - Express Node.js
 - Tailwind CSS 
 - Rasberry Pico Pi 
 - Python  

# Install dependencies

$ npm install

$ node index.js

$ connect your rasberry

# Run the project

$ start index
